## 使用说明

1. **找核心表**：先确定业务主题，在对应分类下查找
   - 渠道+产品+销量 → `全业务资料表`（69）
   - 收入分析 → `基本面月清单`（70）或`全量科目级收入`（89）

2. **补字段**：核心表确定后，再从收入表匹配收入字段，从维表补名称

3. **Hive表名**：待补充
