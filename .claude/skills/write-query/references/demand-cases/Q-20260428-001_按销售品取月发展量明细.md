# 按销售品取月发展量明细（含揽装、划小、是否竣工）

## 业务说明

按用户给定的"销售品编码"列表（来自 `dws_offer.prod_offer_code`），统计指定月份的发展量明细，输出号码、客户名、揽装人、竣工时间、划小局向、揽装局向、销售品名称等字段，并保留"是否竣工"标记列（不过滤）。

**适用场景**：销售品考核明细 / 营销活动跟踪 / 单产品月度发展回看。

## 口径依据

- **发展量** = `action_id IN (1292,6200)`（订购+销售品互换）AND `subs_stat_reason NOT IN ('1200','1300')`（排除撤单作废）AND `subs_stat_date 落区间`
- **是否竣工** = `subs_stat='301200'` 标记列（不进 WHERE，参考 [R-005](../D_experience/cdap_global_rules.md)）
- **销售品名称** ← `dws_offer` JOIN，**必加 `city_id=200`**（参考 [R-001](../D_experience/cdap_global_rules.md)）
- **揽装局向名** ← `dwd_yz_dim_org` JOIN，`levs=3` 分局 / `levs=4` 营服（参考 [R-003](../D_experience/cdap_global_rules.md)）
- **划小局向名** ← 资料表 `dwm_yz_tb_comm_cm_all_final` 按 `serv_id` 关联取 `subst_name/branch_name`

## 参数

| 参数 | 含义 | 示例 |
|------|------|------|
| `offer_codes` | 销售品编码列表 | `'DM0001-884-1-11', 'DM0002-253-1-1'` |
| `month_range` | 竣工时间月份 | `'202509', '202510'` |

## 参考 SQL（多步骤落表风格，对齐生产代码）

```sql
-- 1) 主过滤 + is_jg 标记（销售品 ID 用编码反查）
DROP TABLE IF EXISTS tmp_offer_dev_0;
CREATE TABLE tmp_offer_dev_0
ROW FORMAT DELIMITED FIELDS TERMINATED BY '\u0001'
STORED AS ORC TBLPROPERTIES('orc.compression'='snappy') AS
SELECT
    CASE WHEN o.subs_stat = '301200' THEN 1 ELSE 0 END AS is_jg,
    o.subs_stat,
    o.acc_nbr,
    o.serv_id,
    o.cust_name,
    o.prod_offer_id,
    o.sales_code,
    o.sales_man_name,
    o.salestaff_subst_id,
    o.salestaff_branch_id,
    o.subs_stat_date,
    o.act_date
FROM dwm_yz_rpt_comm_ba_msdisc_final o
WHERE o.prod_offer_id IN (
        SELECT offer_id
        FROM dws_crm_cfguse.dws_offer
        WHERE city_id = 200
          AND prod_offer_code IN (
              'DM0001-884-1-11','DM0001-884-1-12','DM0001-884-1-13',
              'DM0001-884-1-21','DM0001-884-1-31','DM0001-884-1-32',
              'DM0002-253-1-14','DM0002-253-1-12','DM0002-253-1-11',
              'DM0002-253-1-4','DM0002-253-1-2','DM0002-253-1-1'
          )
      )
  AND COALESCE(o.subs_stat_reason, '-1') NOT IN ('1200','1300')   -- R-002 排除撤单作废
  AND date_format(o.subs_stat_date, 'yyyyMM') IN ('202509','202510')
  AND o.action_id IN (1292, 6200);                                  -- 1292 订购 / 6200 销售品互换

-- 2) 关联资料表取划小局向
DROP TABLE IF EXISTS tmp_offer_dev_1;
CREATE TABLE tmp_offer_dev_1
ROW FORMAT DELIMITED FIELDS TERMINATED BY '\u0001'
STORED AS ORC TBLPROPERTIES('orc.compression'='snappy') AS
SELECT a.*, b.subst_name, b.branch_name
FROM tmp_offer_dev_0 a
LEFT JOIN (
    SELECT serv_id, subst_name, branch_name
    FROM dwm_yz_tb_comm_cm_all_final
    WHERE par_month_id = 202509
) b ON a.serv_id = b.serv_id;

-- 3) 关联销售品维表取销售品编码/名称（city_id=200 必加，R-001）
DROP TABLE IF EXISTS tmp_offer_dev_2;
CREATE TABLE tmp_offer_dev_2
ROW FORMAT DELIMITED FIELDS TERMINATED BY '\u0001'
STORED AS ORC TBLPROPERTIES('orc.compression'='snappy') AS
SELECT a.*, b.prod_offer_code, b.offer_name
FROM tmp_offer_dev_1 a
LEFT JOIN (
    SELECT offer_id, prod_offer_code, offer_name
    FROM dws_crm_cfguse.dws_offer
    WHERE city_id = 200
) b ON a.prod_offer_id = b.offer_id;

-- 4) 关联机构维表取揽装局向（levs=3 分局 / levs=4 营服，R-003）
DROP TABLE IF EXISTS tmp_offer_dev_3;
CREATE TABLE tmp_offer_dev_3
ROW FORMAT DELIMITED FIELDS TERMINATED BY '\u0001'
STORED AS ORC TBLPROPERTIES('orc.compression'='snappy') AS
SELECT
    a.*,
    b.org_name AS lz_subst_name,
    c.org_name AS lz_branch_name
FROM tmp_offer_dev_2 a
LEFT JOIN dwd_yz_dim_org b
       ON a.salestaff_subst_id  = b.org_id AND b.levs = 3
LEFT JOIN dwd_yz_dim_org c
       ON a.salestaff_branch_id = c.org_id AND c.levs = 4;        -- 注意 c.levs，避免 AP-009 笔误

-- 5) 最终输出
DROP TABLE IF EXISTS ads_yz_zqb_xyqg;
CREATE TABLE ads_yz_zqb_xyqg
ROW FORMAT DELIMITED FIELDS TERMINATED BY '\u0001'
STORED AS ORC TBLPROPERTIES('orc.compression'='snappy') AS
SELECT
    is_jg,
    subs_stat,
    acc_nbr,
    serv_id,
    cust_name,
    prod_offer_id,
    sales_code,
    sales_man_name,
    subs_stat_date,
    act_date,
    subst_name,
    branch_name,
    prod_offer_code,
    offer_name,
    lz_subst_name,
    lz_branch_name
FROM tmp_offer_dev_3;
```

## 易错点（已沉淀进 D 层）

- **不要选燃气卫士/视联网/FTTR 等专项表**：参见 [AP-001](../D_experience/anti_patterns.md)
- **`dws_offer` 必加 `city_id=200`**：参见 [R-001](../D_experience/cdap_global_rules.md)
- **撤单作废排除不能漏**：参见 [R-002](../D_experience/cdap_global_rules.md)
- **机构维表用 `org_id` + `levs`，不要用 `subst_id`**：参见 [R-003](../D_experience/cdap_global_rules.md)
- **`is_jg` 是标记列不是 WHERE 条件**：参见 [R-005](../D_experience/cdap_global_rules.md)
- **多次 JOIN 时 `levs` 限定要写在本次 JOIN 别名上**：参见 [AP-009](../D_experience/anti_patterns.md)
- **生产表名以现网为准，041 无 `zone_gz_yz` 前缀，资料表是 `dwm_` 不是 `ads_`**：参见 [R-004](../D_experience/cdap_global_rules.md)
