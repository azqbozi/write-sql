# 按营服汇总主宽与 129+ 宽带入网

## 业务说明

按划小**营服**（`branch_name` / `branch_id`）汇总一个月内的 **主宽入网**与 **129+ 宽带入网** 两类结果，用于分公司横向对比。具体筛选条件**不得自行发明**，须从指标技术口径复用。

## 口径依据

- **主宽入网数**、**129+宽带入网数**：以 [M-BASIC-BB-001](../metrics/基本面/M-BASIC-BB-001_主宽入网数.md) 与 [M-BASIC-BB-004](../metrics/基本面/M-BASIC-BB-004_129+宽带入网数.md) 中 **技术口径 (SQL)** 为准（主宽多来自 `view_ads_yz_kd_new_list` 或同类视图；129+ 定义见同文件内对应小节）。  
- 全业务资料表上与本主题相关的**条件速查**见 [tables/069_全业务资料表.md](../tables/069_全业务资料表.md)「本表常用条件速查」；**与指标 SQL 冲突时以指标 SQL 为准**。

- 主宽与 129+ **可能**基于不同基表/视图，不要强行单表 `COUNT` 混写而不核对指标。  
- `par_month_id` 类型与比较方式（`=` / `BETWEEN`）与现网一致。  
- 营服字段以实际口径所用表中的 `branch_id` / `branch_name` 为准。
